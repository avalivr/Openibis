# Openibis
Overview

The openibis algorithm is an open reimplementation of the proprietary BIS (Bispectral Index) depth-of-anesthesia algorithm.This project translated the original MATLAB implementation of openibis into Python using NumPy and SciPy, then validated the Python implementation against reference BIS scores recorded from 24 clinical cases.

Requirements

Jupyter Notebook
